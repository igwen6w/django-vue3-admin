## 获取市中心工单系统登录状态

### 工作流程图

flowchart TD
    A{检查本地登录态是否有效?} -->|有效| B[返回登录态给调用方]
    A -->|无效或过期| C[请求验证码URL<br>获取验证码图片和初始Cookie]
    C --> D[保存初始Cookie<br>解码验证码]
    D --> E[携带Cookie + 账号 + 验证码<br>发送登录请求]
    E --> F{登录成功?}
    F -->|是| G[保存最新Cookie<br>持久化到Redis/DB]
    G --> H[返回登录态给调用方]
    F -->|否| I[登录失败<br>重试登录,有限次数]
    I --> E
